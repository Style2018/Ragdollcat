布偶貓貓舍網站 - 使用說明
=====================================

📁 網站文件結構：
F:\Project\Ragdollcat\
├── index.html                 # 主頁（關於貓舍）
├── css/
│   └── style.css              # 網站樣式表
├── js/
│   └── script.js              # 網站功能腳本
├── images/                    # 圖片文件夾（可放置貓咪照片）
├── pages/                     # 各個子頁面
│   ├── health-guarantee.html  # 貓舍健康保證
│   ├── breeding-cats.html     # 種公/種母
│   ├── adoption-process.html  # 選貓流程＆預約注意事項
│   ├── pricing.html          # 讓渡價格區分
│   ├── find-kitten.html      # 貓寶貝找家中
│   ├── faq.html              # 問與答 Q&A
│   └── competition.html      # 參賽
└── README.txt                # 本說明文件

🌐 如何使用：
1. 直接雙擊 index.html 在瀏覽器中開啟網站
2. 或者將整個資料夾放置在網頁伺服器中訪問

✨ 網站特色：
- 響應式設計，支援手機、平板、電腦
- 優雅的布偶貓風格配色
- 完整的貓舍功能頁面
- 左側固定導航選單
- FAQ 問答互動功能
- 平滑滾動效果
- 手機版選單功能

🎨 設計風格：
- 主色調：優雅的米色、灰色系
- 強調色：金色 (#d4af37)
- 字體：微軟正黑體為主
- 佈局：左側導航 + 右側內容

📝 內容客製化：
- 修改 HTML 文件中的文字內容
- 將貓咪照片放入 images 資料夾
- 更新 HTML 中的圖片路徑
- 修改聯絡資訊和價格

🔧 技術說明：
- 純 HTML + CSS + JavaScript 靜態網站
- 不需要後端伺服器或資料庫
- 相容所有現代瀏覽器
- 支援 SEO 優化

📱 手機版功能：
- 自動隱藏側邊欄
- 漢堡式選單按鈕
- 觸控友善的介面
- 響應式圖片和佈局

🎯 後續優化建議：
